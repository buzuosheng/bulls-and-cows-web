# Bulls & Cows — 单人模式规则

## 游戏简介

电脑随机生成一个四位秘密数字，玩家通过逐步猜测与逻辑推理，尝试用最少的次数猜出这个数字。

---

## 数字规则

### 答案

- 由 **4 位**数字组成，每位为 0～9
- 四位数字**互不重复**
- **首位可以是 0**（例如 0731 是合法的答案）

### 猜测

- 同样需要输入 **4 位**数字
- 猜测时**允许重复**数字（例如 1123 是合法的猜测）
- 首位同样可以是 0

---

## 游戏流程

1. 游戏开始，电脑自动生成一个四位不重复的秘密数字
2. 玩家输入一个四位猜测（可通过屏幕数字键盘或物理键盘输入）
3. 每次猜测后，右侧显示本次的**命中数**，格子颜色不变
4. 玩家根据命中数持续推理，直到命中数为 **4** 即获胜
5. 最多可猜 **10 次**，超过则游戏结束

---

## 命中数的计算方式

**命中数 = 数字正确 且 位置也正确的位数**

猜测与答案逐位对比，该位数字完全一致才计入命中。

**示例（秘密数字为 9375）：**

| 猜测 | 命中数 | 说明 |
|------|--------|------|
| 9753 | 1 | 仅第一位 9 位置正确 |
| 9300 | 2 | 第一位 9、第二位 3 位置正确 |
| 5937 | 0 | 四个数字都在答案中，但没有一位位置正确 |
| 1234 | 0 | 没有任何一位位置正确 |
| 9375 | 4 | 完全正确，获胜 🎉 |

> ⚠️ **数字存在但位置不对，不计入命中数。**

---

## 界面反馈

- 每行猜测提交后，**右侧显示命中数**，颜色随命中数变化：

| 命中数 | 颜色 |
|--------|------|
| 0 | 灰色 |
| 1 | 琥珀黄 |
| 2 | 橙色 |
| 3 | 浅绿 |
| 4 | 深绿 |

- 格子本身**不变色**，无论命中多少位
- 命中数为 **4** 时，四个格子全部变为**绿色**，并触发庆祝动画

---

## 胜负判定

- 命中数达到 **4**：游戏结束，**玩家获胜** 🎉
- 猜测次数超过 **10 次**：游戏结束，**挑战失败**，揭示答案

## 输入方式

| 操作 | 屏幕键盘 | 物理键盘 |
|------|----------|----------|
| 输入数字 | 点击数字按钮 | 数字键 0–9 |
| 删除 | 点击 ← 按钮 | Backspace |
| 提交猜测 | 点击 ✓ 按钮 | Enter |

> 💡 使用 Vimium 等浏览器键盘插件时，页面加载后会自动进入可输入状态，无需手动切换模式。

## 推理示例

```
秘密数字：9375

第 1 次：0123 → 命中 0  排除 0、1、2、3
第 2 次：4567 → 命中 0  排除 4、5、6、7
第 3 次：9800 → 命中 1  确认 9 或 8 有一位正确
第 4 次：9375 → 命中 4  🎉 获胜
```
